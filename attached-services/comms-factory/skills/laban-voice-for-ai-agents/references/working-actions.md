# Working Actions reference

The 8 visible Working Actions Mirodan inherits from Laban. Each is a 3-axis combination of Weight × Space × Time. They split into two columns by Time:

```
SUSTAINED (introvert / preparation)     QUICK (extravert / release)
─────────────────────────────────────────────────────────────────
Pressing  (Strong / Direct / Sus)       Punching   (Strong / Direct / Q)
Wringing  (Strong / Flexible / Sus)     Slashing   (Strong / Flexible / Q)
Gliding   (Light / Direct / Sus)        Dabbing    (Light / Direct / Q)
Floating  (Light / Flexible / Sus)      Flicking   (Light / Flexible / Q)
```

## Preparation hierarchy (Mirodan §1.7, p. 347)

> "We cannot carry out a Punching action without first undergoing the inner preparation of Pressing."

Every Quick action requires its Sustained partner as preparation. Without prep, the Quick action **degrades** to its Sustained version.

| Quick | Required prep | If missing, degrades to |
|-------|---------------|------------------------|
| Punching | Pressing | Pressing |
| Slashing | Wringing | Wringing |
| Dabbing | Gliding | Gliding |
| Flicking | Floating | Floating |

**Implication for caption writing:** a post that opens cold with a Quick-action line ("X IS LIVE.") feels flat — it's trying to Punch without Pressing first. The Sombre/Practical/Commanding opener IS the prep for the Quick-action close. Pair them.

## Subconscious Motifs — what each tempo defaults to

Tempi map to Working-Action motor pairs. Each tempo's "feel" comes from its motor.

### Stable tempi (Inner = Weight + Space, no Flow axis, no Fusions)
- **Commanding** (Strong/Direct): Pressing → Punching — bold resolve
- **Receptive** (Light/Flexible): Floating → Flicking — welcoming tenderness
- **Practical** (Strong/Flexible): Wringing → Slashing — spell-binding power
- **Self-Contained** (Light/Direct): Gliding → Dabbing — gentle deference

### Adream tempi (Inner = Weight + Flow, Fusions present)
- **Sombre** (Strong/Bound): Pressing → Punching — overpowering sombre awareness
- **Irradiant** (Light/Free): Floating → Flicking — sympathetic exultation
- **Overpowering** (Strong/Free): Wringing → Slashing — irradiant casting of a spell
- **Diffused** (Light/Bound): Gliding → Dabbing — diffused welcome feeling

### Near tempi (Inner = Weight + Time, no Fusions)
- **Materialistic** (Strong/Quick): Punching → Slashing — sudden aggressive intent
- **Human** (Light/Sustained): Floating → Gliding — slow tenderness decision
- **Warm** (Strong/Sustained): Pressing → Wringing — warm consent
- **Cool** (Light/Quick): Dabbing → Flicking — pert intimacy

### Mobile (outer of Adream Time / Near Flow only — never baseline)
- **Unacknowledged** (Quick/Bound): Punching / Slashing
- **Acknowledged** (Sustained/Free): Floating / Gliding
- **Revealed** (Quick/Free): Dabbing / Flicking
- **Concealed** (Sustained/Bound): Pressing / Wringing

### Remote (outer of Adream Space / Stable Flow only — never baseline)
- **Egocentric** (Direct/Bound): Pressing → Punching — unsociable solitude
- **Altruistic** (Flexible/Free): Floating → Flicking — sociable cordiality
- **Sociable** (Direct/Free): Gliding → Dabbing — sociable companionship
- **Unsociable** (Flexible/Bound): Wringing → Slashing — welcome solitude

### Awake (outer of Stable Time / Near Space only — never baseline)
- **Acute** (Direct/Quick): Punching → Dabbing — acute idea-decision
- **Doubting** (Flexible/Sustained): Floating → Wringing — slowly dawning doubt
- **Certain** (Direct/Sustained): Pressing → Gliding — certain awareness
- **Uncertain** (Flexible/Quick): Flicking → Slashing — sudden new idea

## Fusions (Flow axis only)

Per Mirodan §1.1 (the "Free pole is fake" override): Free Flow appears only as a *mask* covering Bound Flow inside. Fusion is the dramatic device of *transparent self-deception*. Only the Flow axis has Fusion equivalents — the other axes are visible-as-they-are.

Examples:
- Strong/Bound going Strong/Free → Pressing going Slashing (adolescent rage held in, releasing as slap)
- Light/Bound going Light/Free → Gliding going Flicking (corpse-and-recover, "playing puzzled")

For voice work, Fusions matter when writing posts that *appear* relaxed/casual but have anxiety underneath — the validator should detect this when present.
