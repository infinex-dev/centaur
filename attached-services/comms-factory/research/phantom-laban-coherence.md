## 5. Methodology notes

- Classifications via `scripts/classify-corpus.ts` (Sonnet 4.6, batched, system prompt teaches the Mirodan framework with 5 worked literary anchors).
- Cluster tightness column: 0.00 = perfectly clustered (one value covers everything), 1.00 = perfectly uniform (no dominant value). Below 0.50 = clear modal placement; above 0.70 = scattered / multi-register.
- Drive-family share: a sample counts as Doing-Passion family if all its legible drives (primary + secondary) are in `{doing, passion}`; same for Spell-Vision. Mixed samples are reported separately.
